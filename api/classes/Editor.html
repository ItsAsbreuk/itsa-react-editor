minHeight

String

Defined in lib/component.jsx:74

Available since 15.0.0

To specify a minimum height of the editable area. You best need this property, because min-height cannot be set on the Component-class (this doesn't work). Using classes would mean you'll need to set a deeper div where you would need nested classes, to prevent all other editors having the same min-height.
